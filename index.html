Comenzando en HTML

¿Qué es HTML?

HyperText Markup Language (HTML) es un lenguaje de marcado (no de programación) utilizado para crear páginas web. Este tipo de código te permitirá crear una página web sencilla.

Al abrir un HTML mal construido nos puede mostrar lo siguiente:

  • Una página de error
  • Que el navegador se quede en blanco
  • La página tal y como esperábamos sin aparente error
